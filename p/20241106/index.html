Homebrew インストール

1

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

~/.config/fish/config.fish  に、PATH を通すためのコマンドを追加 fish 起動時に実行されるもの。

config.fish

1
2
3

if status is-interactive
    eval (/opt/homebrew/bin/brew shellenv)
end

ターミナルを再起動

Starship インストール

1

$ brew install starship

starship init fish | source を ~/.config/fish/config.fishに追加

fisher インストール

1

curl -sL https://git.io/fisher | source && fisher install jorgebucaran/fisher

color theme 変更

https://github.com/Jomik/fish-gruvbox

1

fisher install jomik/fish-gruvbox

theme_gruvbox dark hard を ~/.config/fish/config.fishに追加

{
  "terminal.integrated.profiles.osx": {
    "fish": {
      "path": "/opt/homebrew/bin/fish",
      "args": ["-l"]
    }
  },
  "terminal.integrated.defaultProfile.osx": "fish"
}
